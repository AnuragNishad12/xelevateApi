Aircraft API Documentation

API for managing aircraft data including adding, updating, and deleting aircrafts like the "Global 6000".

Overview

The Aircraft API allows users to manage and retrieve aircraft details such as name, images, specifications, and more. It supports operations like GET, POST, PATCH, and DELETE to manage aircraft data efficiently.

API Endpoints

GET /api/aircrafts

Returns a list of all aircrafts.

Response

[
  {
    "id": "1",
    "name": "Global 6000",
    "images": ["image1_url", "image2_url"],
    "shortDescription": "Description of the Global 6000",
    "aircraftDetails": {
      "guestCapacity": 15,
      "numberOfPilots": 2,
      "numberOfFlightAttendants": 1
    },
    "technicalSpecifications": {
      "exterior": {
        "length": "99.5 ft",
        "wingspan": "94 ft"
      },
      "speed": {
        "highSpeed": "944 Km/Hr"
      }
    }
  }
]

GET /api/aircrafts/:id

Returns details of a specific aircraft by its ID.

Parameter: id (string) - Aircraft ID

  "name": "Global 6000",
  "images": ["image1_url", "image2_url"],

    "numberOfPilots": 2,
    "numberOfFlightAttendants": 1

      "length": "99.5 ft",
      "wingspan": "94 ft"
    },
    "speed": {
      "highSpeed": "944 Km/Hr"
    }
  }
}

PATCH /api/aircrafts/:id

Updates details of an existing aircraft by ID.

Parameter: id (string) - Aircraft ID

Request Body

{
  "name": "Updated Global 6000",
  "images": ["updated_image1_url", "updated_image2_url"]
}

Response

{
  "id": "1",
  "name": "Updated Global 6000",
  "images": ["updated_image1_url", "updated_image2_url"],
  "shortDescription": "Updated description",
  "aircraftDetails": {
    "guestCapacity": 15,
    "numberOfPilots": 2,
    "numberOfFlightAttendants": 1
  },
  "technicalSpecifications": {
    "exterior": {
      "length": "99.5 ft",
      "wingspan": "94 ft"
    },
    "speed": {
      "highSpeed": "944 Km/Hr"
    }
  }
}

DELETE /api/aircrafts/:id

Deletes a specific aircraft by ID.

Parameter: id (string) - Aircraft ID

Response

Status: 204 No Content

DELETE /api/aircrafts

Deletes all aircrafts from the database.

Response

Status: 204 No Content
