Customer Review API Documentation

This API allows you to manage customer reviews including creating, updating, retrieving, and deleting reviews.

Base URL

http://localhost:5000/api/customer-reviews

API Endpoints

GET /api/customer-reviews

Returns all customer reviews.

Request: No parameters needed.

Response: A list of all customer reviews.

GET /api/customer-reviews/:id

Returns a specific customer review by ID.

Request: Pass the review ID in the URL.

GET /api/customer-reviews/c1f7badd-4e3f-4c4e-a42e-f5a35fd8775c

Response: A single customer review.

POST /api/customer-reviews

Adds a new customer review. A unique ID is automatically generated.

Request: A JSON object representing the new review. The `id` is automatically generated.

{
  "name": "John Doe",
  "rating": 5,
  "text": "Amazing service! The jersey I ordered was perfect and arrived on time. Highly recommend this company!",
  "ProfileImage": "https://plus.unsplash.com/premium_photo-1671656349322-41de944d259b?q=80&w=1974&auto=format&fit=crop&ixlib=rb-4.0.3&ixid=M3wxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8fA%3D%3D"
}

Response: The created review object with an automatically generated ID.

PATCH /api/customer-reviews/:id

Updates a specific customer review by ID.

Request: A JSON object with the fields to update. Pass the review ID in the URL.

PATCH /api/customer-reviews/c1f7badd-4e3f-4c4e-a42e-f5a35fd8775c
{
  "text": "Updated review! This service was even better the second time around."
}

Response: The updated review object.

{
  "id": "c1f7badd-4e3f-4c4e-a42e-f5a35fd8775c",
  "name": "John Doe",
  "rating": 5,
  "text": "Updated review! This service was even better the second time around.",
  "ProfileImage": "https://plus.unsplash.com/premium_photo-1671656349322-41de944d259b?q=80&w=1974&auto=format&fit=crop&ixlib=rb-4.0.3&ixid=M3wxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8fA%3D%3D"

DELETE /api/customer-reviews/:id

Deletes a specific review by its ID.

Request: No body is needed. Pass the review ID in the URL.

DELETE /api/customer-reviews/c1f7badd-4e3f-4c4e-a42e-f5a35fd8775c

Response: Status code 204 indicating the review was deleted successfully.

DELETE /api/customer-reviews

Deletes all customer reviews.

Request: No body is needed. This will delete all reviews in the file.

Response: Status code 204 indicating all reviews were deleted successfully.
