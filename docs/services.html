Our Services API Documentation

Comprehensive guide to interacting with the "Our Services" API

Overview

This API allows you to manage "Our Services" by performing CRUD operations (Create, Read, Update, Delete). It supports UUID-based identifiers for each service entry.

Base URL

The base URL for all API requests is:

https://yourapi.com/api/our-services

API Endpoints

GET /api/our-services

Method: GET

Fetch all services available in the database.

GET /api/our-services

Response:
  200 OK
  [
    {
      "id": "12345678-abcd-1234-abcd-1234abcd1234",
      "name": "Web Development",
      "image": "https://example.com/image.jpg",
      "description": "Custom web development services"
    },
    ...
  ]

GET /api/our-services/:id

Method: GET

Retrieve a specific service by its UUID.

GET /api/our-services/12345678-abcd-1234-abcd-1234abcd1234

Response:
  200 OK
  {
    "id": "12345678-abcd-1234-abcd-1234abcd1234",
    "name": "Web Development",
    "image": "https://example.com/image.jpg",
    "description": "Custom web development services"
  }

POST /api/our-services

Method: POST

Create a new service and automatically generate a UUID for it.

POST /api/our-services

Request Body:
  {
    "name": "Mobile App Development",
    "image": "https://example.com/app.jpg",
    "description": "Building high-quality mobile apps"
  }

Response:
  201 Created
  {
    "id": "87654321-dcba-4321-dcba-4321abcd4321",
    "name": "Mobile App Development",
    "image": "https://example.com/app.jpg",
    "description": "Building high-quality mobile apps"
  }

PATCH /api/our-services/:id

Method: PATCH

Update an existing service by its UUID.

PATCH /api/our-services/12345678-abcd-1234-abcd-1234abcd1234

Request Body:
  {
    "name": "Updated Service Name",
    "image": "https://example.com/new-image.jpg",
    "description": "Updated description for the service"
  }

Response:
  200 OK
  {
    "id": "12345678-abcd-1234-abcd-1234abcd1234",
    "name": "Updated Service Name",
    "image": "https://example.com/new-image.jpg",
    "description": "Updated description for the service"
  }

DELETE /api/our-services/:id

Method: DELETE

Delete a specific service by its UUID.

DELETE /api/our-services/12345678-abcd-1234-abcd-1234abcd1234

Response:
  204 No Content

DELETE /api/our-services

Method: DELETE

Delete all services in the database.

DELETE /api/our-services

Response:
  204 No Content
Note: All API responses are in JSON format, and the request body must be sent in JSON format as well.
