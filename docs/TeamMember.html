Team Member API Documentation

Learn how to interact with the Team Member API

API Overview

The Team Member API allows you to perform CRUD operations (Create, Read, Update, Delete) for managing team members. You can access team member data, add new members, update existing ones, or delete them.

GET /api/team-members

Description:

Returns a list of all team members.

Request:

GET /api/team-members

GET /api/team-members/:id

Description:

Returns a specific team member by ID.

Request:

GET /api/team-members/:id

Response:

{
  "id": "1",
  "name": "Kanika Tekriwal",
  "role": "Founder",
  "image": "https://images.unsplash.com/photo-1500648767791-00dcc994a43e?q=80&w=1974&auto=format&fit=crop&ixlib=rb-4.0.3&ixid=M3wxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8fA%3D%3D"

POST /api/team-members

Description:

Creates a new team member and automatically generates a UUID.

Request:

POST /api/team-members

Request Body:

{
  "name": "Hinata",
  "role": "Co-Founder",
  "image": "https://via.placeholder.com/150"
}

Response:

{
  "id": "5f1a21c7-e7f9-4c0d-81db-bcd6175b23f4",
  "name": "Hinata",
  "role": "Co-Founder",
  "image": "https://via.placeholder.com/150"
}

PATCH /api/team-members/:id

Description:

Updates an existing team member by ID.

Request:

PATCH /api/team-members/:id

Request Body:

{
  "name": "Hinata Uzumaki",
  "role": "Co-Founder",
  "image": "https://via.placeholder.com/150"
}

Response:

{
  "id": "5f1a21c7-e7f9-4c0d-81db-bcd6175b23f4",
  "name": "Hinata Uzumaki",
  "role": "Co-Founder",
  "image": "https://via.placeholder.com/150"
}

DELETE /api/team-members/:id

Description:

Deletes a specific team member by ID.

Request:

DELETE /api/team-members/:id

Response:

204 No Content

DELETE /api/team-members

Description:

Deletes all team members.

Request:

DELETE /api/team-members

Response:

204 No Content
