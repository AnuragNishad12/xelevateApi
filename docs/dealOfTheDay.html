Deal of the Day API Documentation

This API allows you to manage "deal of the day" data with various operations including creating, updating, retrieving, and deleting deals.

Base URL

http://localhost:5000/api/deal-of-the-day

Endpoints

GET /api/deal-of-the-day

Returns a list of all deals.

Request: No parameters needed.

Response: An array of deals (if any).

[
  {
    "id": "uuid",
    "name": "Mountain View Chalet",
    "location": "Alps, Switzerland",
    "rating": "9.2",
    "ratingText": "Exceptional",
    "reviewCount": "1,824",
    "price": "18,750",
    "imageUrl": "https://images.unsplash.com/photo-1581819896533-f8ab6767ce7e?q=80&w=2101&auto=format&fit=crop&ixlib=rb-4.0.3&ixid=M3wxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8fA%3D%3D"
  }
]

GET /api/deal-of-the-day/:id

Returns a specific deal by its ID.

Request:

  • id: The unique ID of the deal.

Response: A single deal object or error message.

{
  "id": "uuid",
  "name": "Mountain View Chalet",
  "location": "Alps, Switzerland",
  "rating": "9.2",
  "ratingText": "Exceptional",
  "reviewCount": "1,824",
  "price": "18,750",
  "imageUrl": "https://images.unsplash.com/photo-1581819896533-f8ab6767ce7e?q=80&w=2101&auto=format&fit=crop&ixlib=rb-4.0.3&ixid=M3wxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8fA%3D%3D"
}

POST /api/deal-of-the-day

Adds a new deal to the system.

Request: A JSON object representing the new deal.

{
  "name": "New Mountain Retreat",
  "location": "Rocky Mountains, Canada",
  "rating": "9.8",
  "ratingText": "Excellent",
  "reviewCount": "2,345",
  "price": "25,000",
  "imageUrl": "https://example.com/new-image.jpg"
}

Response: The created deal object.

{
  "id": "new-uuid",
  "name": "New Mountain Retreat",
  "location": "Rocky Mountains, Canada",
  "rating": "9.8",
  "ratingText": "Excellent",
  "reviewCount": "2,345",
  "price": "25,000",
  "imageUrl": "https://example.com/new-image.jpg"
}

PATCH /api/deal-of-the-day/:id

Updates a specific deal by its ID.

Request: A JSON object with the updated fields for the deal.

{
  "name": "Updated Mountain Retreat",
  "price": "20,000"
}

Response: The updated deal object.

{
  "id": "updated-uuid",
  "name": "Updated Mountain Retreat",
  "location": "Rocky Mountains, Canada",
  "rating": "9.8",
  "ratingText": "Excellent",
  "reviewCount": "2,345",
  "price": "20,000",
  "imageUrl": "https://example.com/updated-image.jpg"
}

DELETE /api/deal-of-the-day/:id

Deletes a specific deal by its ID.

Request: No body is needed. The deal will be deleted based on the ID provided in the URL.

Response: A status code of 204 indicating the deal was successfully deleted.

DELETE /api/deal-of-the-day

Deletes all deals in the system.

Request: No body is needed. This will delete all deals in the file.

Response: A status code of 204 indicating all deals were successfully deleted.
